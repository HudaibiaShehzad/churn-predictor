# Customer Churn Prediction ML Pipeline

## Live Demo
[Try the app here](https://churn-predictor-hudaibia.streamlit.app)

## Overview
End-to-end ML pipeline predicting customer churn for a telecom company.

## Results
| Model | ROC-AUC | F1 |
|---|---|---|
| Logistic Regression | 0.8467 | 0.5935 |
| Gradient Boosting | 0.8422 | 0.5644 |
| Random Forest | 0.8224 | 0.5438 |
| SVM | 0.7916 | 0.5589 |

## Tech Stack
- Python, scikit-learn, SHAP, MLflow, Streamlit

## What I Built
- EDA with 3 insight charts
- 4 ML models trained and compared
- MLflow experiment tracking
- SHAP explainability
- Live deployed web app
